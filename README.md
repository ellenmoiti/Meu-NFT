# Meu-NFT

## Links úteis
- https://opensea.io/
- https://metamask.io/

## Passo a passo
1) Conecte-se com o Metamask
2) Conecte-se no Opensea via Metamask
3) No Opensea:
   - Create -> Collection or Item
   - Preencha as informações solicitadas.
   - Certifique-se de ter em sua conta o valor necessário para a criação do NFT.
